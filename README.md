# 💼 Job Portal System

A full-stack Job Portal web application built using Flask and SQLite.  
Users can browse jobs and apply with resume upload, while admin can post jobs and manage applications.

---

## 🚀 Features

### 👤 User
- Signup & Login system
- View available jobs
- Apply for jobs with resume upload
- Track applied jobs (My Applications)

### 👑 Admin
- Add new job postings
- View all user applications
- Download user resumes
- Manage job listings

---

## 🛠️ Tech Stack

- Backend: Flask (Python)
- Database: SQLite
- Frontend: HTML, CSS (Responsive UI)
- Templating: Jinja2
